## Features

- **Clean Structure**: Well-organized semantic HTML5 elements
- **Responsive**: Built with mobile-first viewport meta tag
- **Fast Loading**: No external dependencies or heavy frameworks
- **Professional Layout**: Logical flow of information from summary to details
- **Integrated Links**: Direct links to GitHub and LinkedIn profiles
- **Updated Content**: Current certifications and education timeline

## Installation & Usage

### Quick Start
1. **Download the file**: Save `index.html` to your computer
2. **Open in browser**: Double-click the file or right-click → Open With → Browser
3. **View resume**: The complete resume will display in your default browser

### For Development
bash
# Clone or download the HTML file
# Open with preferred editor (VS Code, Sublime Text, etc.)
# Make modifications as needed
# Refresh browser to see changes
